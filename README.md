## 介绍

一款基于 Vue 的移动端宁波疫情动态监控系统，结合电子地图、影像地图和各种常见的图表(柱状图、折线图、饼状图)，实现数据与地图的联动显示，更直观形象的展示除了数据指标变化。
tips：本站数据都是爬取的真实数据。
另外还：

- GIS 地图展示。
- 移动端页面适配。
- PC 端适配。

## 安装

使用 `git clone` 本项目

**npm install** 安装

**npm run server** 启动 web

**npm start ./server/app.js** 启动 server 服务器即可。

> 数据使用 express 调取。
